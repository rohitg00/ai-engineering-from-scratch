# AI Engineering From Scratch Book Volumes

This folder contains book-style HTML editions of the AI Engineering From Scratch learning material. The goal is to make the same content easier to read as long-form, structured volumes while keeping the interactive website available as the primary learning experience.

## Available Volumes

- `vol-1-foundations.html` - Foundations of AI engineering, including setup, math, classical machine learning, neural networks, computer vision, NLP, and speech/audio.
- `vol-2-modern-ai-and-llms.html` - Modern AI and LLM topics, including transformers, generative models, reinforcement learning, LLM training and alignment, inference, RAG, tool use, multimodal AI, agents, and multi-agent systems.
- `vol-3-ai-in-production.html` - Production AI engineering topics, including serving LLMs at scale, inference economics, deployment, gateways, SRE, security, compliance, FinOps, safety, governance, evaluation, red teaming, and capstone projects.

## Notes

These HTML files are generated from the existing project content and are intended as an additional reading format for learners who prefer book-style material. The website build copies the HTML and CSS files from this folder into `site/book/` so the deployed site can serve the volumes from `/book/<volume-file>.html`.
