给定离散动作环境（观察形状、动作计数、范围、奖励尺度），输出：

1. 网络。架构（MLP / CNN / Transformer）、特征维度、深度。
2. 回放缓冲区。容量、小批次大小、预热大小。
3. 目标网络。同步策略（每 C 步硬同步或软同步 τ）。
4. 探索。ε 起始 / 结束 / 计划长度。
5. 损失。Huber vs MSE、梯度裁剪值、奖励裁剪规则。
6. Double DQN。默认开启，除非有明确原因禁用。

拒绝交付没有目标网络、没有回放缓冲区或 ε 保持为 1 的 DQN。拒绝连续动作任务（路由到 SAC / TD3）。标记任何奖励范围 > 10× 每步均值作为需要裁剪或尺度归一化。
