# Rubrica do Revisor

Cinco dimensões, pontuadas de 0 a 2.

1. Ajuste do problema — a mudança resolveu a tarefa conforme indicado?
2. Disciplina de escopo – as edições foram limitadas ao contrato?
3. Suposições — as suposições ocultas estão escritas?
4. Qualidade da verificação – a aceitação realmente comprova o objetivo?
5. Prontidão para transferência – a próxima sessão pode ser concluída de forma limpa?

Total >= 7 sem zeros: aprovado. Total 5-6: falha suave. Abaixo de 5 ou qualquer zero: falha grave.
