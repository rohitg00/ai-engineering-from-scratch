# AGENTES.md

Você está trabalhando dentro de um repositório que é executado com um ambiente de trabalho do agente.

Leia isto antes de agir:

1. `agent_state.json` — onde a última sessão foi interrompida.
2. `task_board.json` — o que está em voo, o que vem a seguir.
3. `docs/agent-rules.md` — inicialização, proibido, concluído, incerteza, aprovação.
4. `docs/reliability-policy.md` — modos de falha que esta bancada foi projetada para absorver.
5. `docs/handoff-protocol.md` — qual final de sessão deve produzir.
6. `docs/reviewer-rubric.md` — como o trabalho concluído é avaliado.

Comando de verificação: veja `acceptance_criteria` na tarefa ativa no quadro.

Versão do pacote: 1.0.0
