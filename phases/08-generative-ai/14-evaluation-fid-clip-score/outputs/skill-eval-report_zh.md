给定新的生成模型检查点、参考基线和模态（图像 / 视频 / 音频 / 3D），输出完整评估计划：

1. 样本质量。对 10-30k 样本 vs 保留真实集的 FID / FD-DINO / CMMD。匹配分辨率。报告 3 种子均值 +/- 标准差。
2. 遵循度。提示-图像对的 CLIP 分数 / CMMD。包含文本到图像的 HPSv2 + ImageReward + PickScore。对于视频，添加视觉-语言指标（V-Eval）。对于音频，CLAP + MOS。
3. 成对偏好。对 200-2000 提示 vs 基线的盲法 A/B。人类 + LLM 评判 + PartiPrompts 覆盖。
4. 类别细分。每个提示类别的性能（人物、动物、文本渲染、构图、风格）。即使全局指标改善，也标记每个类别的回退。
5. 安全 / 滥用。NSFW 分类器、深度伪造检测器、水印检查、前 K 生成的版权相似度扫描。
6. 签署。明确门槛：FID 在基线 +5% 内或 >55% 人类胜率或记录定性优势。没有单一指标声明。

拒绝报告 N < 5000 的 FID。拒绝交付模型可能在训练中见过的提示上计算的基准。拒绝在没有人工交叉检查的情况下仅报告 LLM 评判结果。标记任何声称指标"上升 20%"而没有报告绝对基值和单一种子的声明。
