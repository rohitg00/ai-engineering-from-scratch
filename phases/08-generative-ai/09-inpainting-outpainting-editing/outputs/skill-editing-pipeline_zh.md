给定源图像、目标编辑（移除 X、用 Z 替换 Y、扩展画布、重风格区域、改变季节 / 一天中的时间）和质量门槛（草稿 / 作品集 / 打印），输出：

1. 掩码策略。显式画笔掩码、SAM 2 点击 / 框提示、文本短语的 Grounded-SAM，或 RMBG（用于背景移除）。一句话说明原因。
2. 基础模型 + 模式。SD-Inpaint / SDXL-Inpaint / Flux-Fill / Flux-Kontext 用于指令编辑，或如果没有掩码则使用 SDEdit 噪声级（0.3 / 0.6 / 0.9）。
3. 提示支架。描述编辑后的整个图像，不只是新内容。包含负面提示。
4. CFG + 强度 + 羽化。掩码羽化 8-16 px；SDXL-inpaint 的 CFG ~5-7，Flux 的 3-4。完全再生强度 0.8-1.0，保留 0.3-0.5。
5. 护栏。NSFW / 深度伪造 / 商标检测钩子、换脸策略门、可逆性（保存掩码 + 种子）。

拒绝在没有明确政策检查的情况下对可识别的公众人物进行身份编辑。拒绝在没有至少 30% 原始画布作为锚点的情况下外扩图像（太少上下文使模型幻觉）。标记任何 t/T > 0.7 且保真度目标为"保留主题"的 SDEdit 运行作为可能不匹配。
