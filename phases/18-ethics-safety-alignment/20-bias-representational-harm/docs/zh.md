# 20 · 大语言模型中的偏见与表征性伤害

> Gallegos、Rossi、Barrow、Tanjim、Kim、Dernoncourt、Yu、Zhang、Ahmed（《计算语言学》Computational Linguistics 2024，arXiv:2309.00770）。这篇 2024 年奠基性综述区分了「表征性伤害（representational harm）」（刻板印象、抹除）与「分配性伤害（allocational harm）」（资源分配不均），并将评估指标归为基于嵌入、基于概率和基于生成文本三类。2024-2025 年实证研究：An 等人（《PNAS Nexus》，2025 年 3 月）在 20 个入门级职位的自动简历评估任务中，测量了 GPT-3.5 Turbo、GPT-4o、Gemini 1.5 Flash、Claude 3.5 Sonnet、Llama 3-70B 在性别与种族交叉维度上的偏见。WinoIdentity（COLM 2025，arXiv:2508.07111）引入了基于不确定性的「交叉性（intersectionality）」公平性评估方法。Yu 与 Ananiadou（2025）在 MLP 层中识别出性别神经元；Ahsan 与 Wallace（2025）利用「稀疏自编码器（sparse autoencoder，SAE）」揭示临床种族偏见；Zhou 等人（2024，UniBias）通过操控注意力头实现去偏。元批判（arXiv:2508.11067）：过去十年的文献过度聚焦于二元性别偏见。

## 学习目标

- 定义表征性伤害与分配性伤害的区别，并各举一个在大语言模型部署中的示例。
- 说出 Gallegos 等人（2024）提出的三类评估指标，并从每类中各描述一个指标。
- 描述交叉性的概念，并说明 WinoIdentity 基于不确定性的公平性测量方法为何能填补单轴偏见评估的空白。
- 描述两种基于「机制可解释性（mechanistic interpretability）」的偏见分析方法（性别神经元、SAE 特征、注意力头操控）。

## 问题

前面几课讨论了蓄意伤害（越狱、策划）与安全治理。偏见则是一种在无意中产生的伤害——源自训练数据分布、提示词框架、以及累积的设计选择。测量和减少偏见在方法论上不同于对抗鲁棒性，有其独特的挑战。

## 概念

### 表征性伤害 vs 分配性伤害

- **表征性伤害。** 刻板印象、抹除、贬低性描绘。如果一个 LLM 将护士一律描绘为女性，就是在产生表征性伤害。
- **分配性伤害。** 不平等的实质性结果。如果一个 LLM 对黑人求职者的简历评分系统性地偏低，就是在产生分配性伤害。

这两者并不等同。一个模型可以在「表征上无偏见」（产生多样化的描绘），同时在「分配上存在偏见」（给出不平等的推荐）。评估需要同时测量两者。

### 三类评估指标（Gallegos 等人，2024）

- **基于嵌入（embedding-based）。** 对 RLHF 之前的嵌入进行 WEAT 风格的测试。测量身份词与属性词之间的统计关联。局限：测量的是表征，而非行为。
- **基于概率（probability-based）。** 比较刻板印象一致补全与刻板印象违背补全的对数似然。解码器侧的测量，能够捕捉部分行为偏见。
- **基于生成文本（generated-text-based）。** 对生成文本进行下游任务测量。简历评分、推荐撰写、对话等。最具生态效度，但最难复现。

### 交叉性

仅按「性别」做偏见评估会遗漏那些仅在（性别，种族）组合上触发的偏见。An 等人（2025）发现 GPT-4o 在简历评分中对黑人女性的惩罚甚于黑人男性，也甚于白人女性——单独按任一轴评估无法捕捉这一现象。

WinoIdentity（COLM 2025）引入了基于不确定性的交叉公平性测量方法。它不仅关注点预测，还测量模型在不同交叉身份元组上的结果不确定性是否存在差异。这能捕捉到模型在各群体上错误程度相同但不确定性不同的情况，而这种差异会导致不同的下游分配行为。

### 机制方法

2024-2025 年的可解释性研究为偏见问题打开了机制性干预的大门：

- **性别神经元（gender neurons，Yu 与 Ananiadou，2025）。** 特定 MLP 神经元与性别相关行为存在相关性。消融这些神经元可在有限的性能代价下缩小性别差距指标。
- **通过 SAE 揭示临床种族偏见（Ahsan 与 Wallace，2025）。** 稀疏自编码器特征将内部表征分解为可解释的维度；与种族相关的特征可被识别并抑制。
- **UniBias（Zhou 等人，2024）。** 通过注意力头操控实现零样本去偏。特定的注意力头会放大对身份类别的敏感性；将这些头归零或重新加权可以在不做微调的情况下减少偏见。

### 元批判

一项对十年文献的综述（arXiv:2508.11067，2025）指出，该领域过度聚焦于二元性别偏见。其他维度——残障、宗教、移民身份、多语言身份——受到的关注少得多。该元批判认为，这种狭窄的聚焦会因忽视而对边缘群体造成伤害：一个在二元性别上去偏效果良好的模型，可能在从未被检查的其他维度上存在严重偏见。

## 关键术语

| 术语 | 人们常说的 | 实际含义 |
|------|-----------------|------------------------|
| 表征性伤害（Representational harm） | 「刻板印象 / 抹除」 | 对一个群体的偏见性描绘 |
| 分配性伤害（Allocational harm） | 「不平等的决策」 | 对一个群体的偏见性实质结果 |
| WEAT | 「那个嵌入测试」 | 词嵌入关联测试（Word Embedding Association Test）；基于共现的偏见探测方法 |
| 交叉性（Intersectionality） | 「组合身份效应」 | 在多个身份轴交叉处显现的偏见 |
| 性别神经元（Gender neurons） | 「MLP 偏见神经元」 | 其激活值与性别相关行为存在相关性的特定神经元 |
| SAE 特征 | 「可解释维度」 | 稀疏自编码器识别的特征；可用于机制性偏见分析 |
| UniBias | 「注意力头去偏」 | 通过重新加权注意力头实现零样本去偏 |
