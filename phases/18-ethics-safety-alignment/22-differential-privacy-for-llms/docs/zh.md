# 大语言模型的差分隐私

> DP-SGD 仍然是标准——注入噪声的梯度更新提供正式的 (ε, δ) 保证。在计算、内存和效用方面开销很大；参数高效差分隐私微调（LoRA + DP-SGD）是常见的 2025 年配置（ACM 2025）。两个证据体处于紧张关系：基于金丝雀的成员推断（Duan et al., 2024）报告对语言模型成功有限；训练数据提取（Carlini et al., 2021; Nasr et al., 2025）恢复大量逐字记忆。解决方案（arXiv:2503.06808，2025 年 3 月）：差距在于测量内容——插入的金丝雀 vs "最可提取的"数据。新的金丝雀设计实现无影子模型的基于损失的 MIA，并产生首个对用真实 DP 保证训练的大语言模型的非平凡 DP 审计。替代方案：PMixED（arXiv:2403.15638）——推理时通过下一令牌分布上的专家混合进行私有预测；DP 合成数据生成（Google Research 2024）。新兴攻击：通过 LLM 反馈的差分隐私逆转——置信度分数泄漏。

## 问题

LLM 记忆。Carlini 等人 2021 表明生产语言模型按需重现逐字训练文本。DP 是正式防御：训练使得输出可证明对单个训练样本不敏感。2024-2025 年证据表明 DP-SGD 是必要的，但部署的 ε 值可能与威胁模型不匹配。

## 概念

### (ε, δ)-差分隐私

随机算法 M 是 (ε, δ)-DP，如果对于任何相差一个样本的数据集和任何事件 S：
P(M(D) ∈ S) ≤ e^ε * P(M(D') ∈ S) + δ。

解释：输出分布足够接近（由 ε 参数化），以至于任何单个个体的贡献不能被可靠推断，除了概率 δ。

### DP-SGD

Abadi et al. 2016。标准方法：
1. 采样小批量。
2. 计算每个样本梯度。
3. 将每个样本梯度裁剪到阈值 C。
4. 求和裁剪后的梯度并添加标准偏差 σ * C 的高斯噪声。
5. 使用噪声和更新参数。

隐私成本由会计师跟踪（矩会计师，Rényi DP 会计师）。LLM 文献中报告的 ε 值因威胁模型、数据敏感性和效用目标而差异很大；没有普遍"安全"默认 ε。已发表的例子大致跨越 ε ≈ 1–10 在某些 LLM 训练设置中，但这些是说明性的——不是推荐的默认值。更低的 ε 通常需要更多噪声，并可能增加效用损失。

### LoRA + DP-SGD

前沿模型的完整 DP-SGD 是禁止性的。LoRA（Hu et al. 2022）将梯度更新限制在小适配器上，减少每个样本梯度存储。LoRA + DP-SGD 是常见的 2025 年配置。DP 保证应用于适配器；基础模型保持固定。

### 2024-2025 年紧张关系

两条证据线：

- **金丝雀 MIA（Duan et al. 2024）。** 将独特金丝雀插入训练数据，测量成员推断攻击者是否能识别它们。报告对语言模型成功有限。表明 MIA 很难。
- **训练数据提取（Carlini 2021，Nasr et al. 2025）。** 用前缀提示模型；测量它是否从训练中恢复逐字文本。报告大量记忆。表明在相关意义上 MIA 很容易。

2025 年 3 月解决方案（arXiv:2503.06808）：两者测量不同事物。MIA 在插入的金丝雀上问"样本 e 在 D 中吗？"提取问"我能从模型恢复 D 的什么？"最可提取的"样本是隐私重要的；金丝雀低估这个，因为它们未被优化为可提取的。

新的金丝雀设计。无影子模型的基于损失的 MIA。对具有真实 DP 保证的 LLM 的首次非平凡 DP 审计。

### DP 训练的替代方案

- **PMixED（arXiv:2403.15638）。** 推理时私有预测。下一令牌分布上的专家混合；每个专家看到训练数据的一个分片；聚合添加噪声以实现 DP。完全避免 DP 训练。
- **DP 合成数据生成（Google Research 2024）。** LoRA 微调与 DP-SGD，采样合成数据，在合成数据上训练下游分类器。

两者都回避完整 DP 训练的效用成本，代价是不同的威胁模型。

### 通过 LLM 反馈的差分隐私逆转

新兴 2025 年攻击。使用 DP 训练的模型的置信度分数作为神谕来重新识别个体。即使输出不泄漏，置信度分布也可以。

防御：不暴露置信度，或在暴露前截断/量化它们。这是超出 (ε, δ)-DP 训练的要求。

## 关键术语

| 术语 | 人们说的 | 实际含义 |
|------|----------|----------|
| DP | "(ε, δ)-差分隐私" | 正式隐私：在相邻数据集变化下输出分布接近 |
| DP-SGD | "噪声注入 SGD" | 梯度裁剪 + 高斯噪声添加；标准 DP 训练 |
| LoRA + DP-SGD | "高效私有微调" | 低秩适配器上的 DP-SGD；常见 2025 年配置 |
| MIA | "成员推断" | 确定样本是否在训练数据中的攻击 |
| 金丝雀 | "插入水印样本" | 用于测量 DP 泄漏的独特训练样本 |
| PMixED | "私有推理混合" | 通过下一令牌分布上的专家混合进行推理时 DP |
| DP 逆转 | "置信度泄漏攻击" | 使用模型置信度作为重新识别的神谕的攻击 |
