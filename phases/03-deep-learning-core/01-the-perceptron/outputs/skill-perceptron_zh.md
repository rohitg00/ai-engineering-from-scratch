# 感知机模式

感知机计算输入的加权和加上偏置，然后应用阶跃函数产生二元输出。它是神经网络的基本单元。

```
output = step(w1*x1 + w2*x2 + ... + wn*xn + bias)
```

## 单层感知机足够的情况

- 问题是线性可分的：一条直线（或超平面）可以分开两类
- 逻辑门：AND、OR、NOT、NAND
- 简单阈值决策："分数是否高于X？"
- 数据聚类成两个不重叠区域的二元分类器

## 需要多层的情况

- 问题不是线性可分的：没有单条直线可以分开类别
- XOR和奇偶校验问题
- 任何需要"这个但不是那个"推理的任务（条件组合）
- 真实世界分类：图像、文本、音频——几乎都是非线性的

## 决策检查清单

1. 绘制或检查你的数据。你能画一条直线边界分开类别吗？
   - 能：单层感知机可行
   - 不能：你至少需要两层
2. 问题能否分解为更简单的线性决策的AND/OR组合？
   - 这种分解告诉你最小网络结构
   - XOR = (A OR B) AND (NOT (A AND B)) = 2层中的3个感知机
3. 对于多于两类的问题，每类需要一个输出节点

## 训练规则

```
error = expected - predicted
weight_new = weight_old + learning_rate * error * input
bias_new = bias_old + learning_rate * error
```

如果预测正确，什么都不变。如果错误，权重调整以减少误差。这只适用于单层感知机。多层网络需要反向传播。

## 常见错误

- 试图用单层感知机学习非线性模式（永远不会收敛）
- 学习率设置太高（权重振荡）或太低（训练永远进行）
- 忘记偏置项（没有它，决策边界必须通过原点）
- 混淆感知机收敛（线性可分数据保证收敛）与一般神经网络收敛（不保证）
