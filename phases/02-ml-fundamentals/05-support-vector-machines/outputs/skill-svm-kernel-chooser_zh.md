# SVM 核函数选择指南

SVM 由两个选择定义：核函数（决定决策边界的形状）和正则化参数（控制间隔宽度与分类错误之间的权衡）。正确设置这些参数是无用模型与强大模型之间的区别。

## 决策清单

1. 数据是否线性可分（或接近线性可分）？
   - 是：使用线性核。它更快且更可解释。
   - 否：进入步骤 2。

2. 特征与样本的比例如何？
   - 特征 >> 样本（如 TF-IDF 文本）：使用线性核。高维数据通常线性可分。RBF 增加复杂度而无收益。
   - 样本 >> 特征（如 10-50 个特征的表格数据）：RBF 核是默认选择。

3. 决策边界预期是平滑的吗？
   - 平滑、连续的边界：RBF 核
   - 多项式形状的边界：多项式核（从 2 或 3 次开始）
   - 领域知识暗示特定的交互项：使用匹配次数的多项式核

4. 数据集有多大？
   - 少于 10,000 个样本：任何核函数都适用，RBF 是安全的默认选择
   - 10,000 到 100,000：线性核或 LinearSVC（原始形式，每轮 O(n)）
   - 超过 100,000：不要使用核 SVM。切换到线性 SVM、梯度提升或神经网络。

5. 你缩放特征了吗？
   - SVM 需要特征缩放。拟合前始终标准化（零均值，单位方差）。未缩放的特征会扭曲间隔几何。

## 核函数选择流程图

```
开始
  |
  v
特征 > 1000 或特征 >> 样本？
  是 --> 线性核（LinearSVC 求速度）
  否  --> 数据集 < 10k 样本？
            是 --> 先尝试 RBF（最佳通用核）
            否  --> 线性核（核 SVM 是 O(n²) 到 O(n³)）
```

如果 RBF 效果不好，尝试 2-3 次多项式。如果也失败，问题可能不适合 SVM。

## 调整 C（正则化）

C 控制误分类的惩罚。它与正则化强度成反比。

| C 值 | 效果 | 何时使用 |
|------|------|----------|
| 0.001 - 0.01 | 宽间隔，允许许多违反 | 噪声数据，想要泛化 |
| 0.1 - 1.0 | 平衡 | 良好的起始范围 |
| 10 - 1000 | 窄间隔，很少违反 | 干净数据，需要高准确率 |

调整策略：
- 从 C=1.0 开始
- 在对数尺度上搜索：[0.001, 0.01, 0.1, 1, 10, 100, 1000]
- 使用交叉验证选择最佳值
- 如果最佳 C 在范围的边缘，向该方向扩展范围

## 调整 gamma（RBF 核）

Gamma 控制单个训练点的影响范围。它定义高斯的宽度。

| gamma 值 | 效果 | 何时使用 |
|----------|------|----------|
| 小 (0.001) | 每个点影响大范围。平滑、简单的边界 | 欠拟合或特征少 |
| 中 (auto: 1/n_features) | sklearn 默认值。合理的起点 | 通用 |
| 大 (10+) | 每个点只影响附近点。复杂、波浪状边界 | 过拟合风险 |

调整策略：
- 从 gamma="scale"（1 / (n_features * X.var())，sklearn 默认值）开始
- 在对数尺度上搜索：[0.001, 0.01, 0.1, 1, 10]
- 低 gamma + 高 C 倾向于过拟合
- 高 gamma + 低 C 倾向于欠拟合

## 联合调整 C 和 gamma

C 和 gamma 相互作用。始终一起调整，而不是独立调整。

推荐方法：
1. 粗网格搜索：C 在 [0.01, 0.1, 1, 10, 100]，gamma 在 [0.001, 0.01, 0.1, 1, 10]（25 种组合）
2. 找到最佳区域
3. 在最佳区域周围进行细网格搜索（如 C 在 [5, 10, 20, 50]，gamma 在 [0.05, 0.1, 0.2]）
4. 全程使用 5 折交叉验证

## 常见错误

- 在高维稀疏数据上使用 RBF 核（线性更好且快 100 倍）
- 忘记缩放特征（最常见的 SVM 错误）
- 在噪声数据上设置 C 太高（记忆噪声而不是学习边界）
- 在超过 50k 样本的数据集上使用核 SVM（训练时间过高）
- 不一起调整 C 和 gamma（它们相互补偿）
- 默认使用 5+ 次多项式（严重过拟合，先尝试 2 或 3 次）

## 快速参考

| 核函数 | 何时使用 | 关键参数 | 训练复杂度 |
|--------|----------|----------|------------|
| 线性 | 文本/TF-IDF，特征多，大数据 | 仅 C | 每轮 O(n) |
| RBF | 通用，少于 10k 样本 | C, gamma | O(n²) 到 O(n³) |
| 多项式 | 已知多项式关系 | C, degree, coef0 | O(n²) 到 O(n³) |
| Sigmoid | 很少有用（等价于两层神经网络） | C, gamma, coef0 | O(n²) 到 O(n³) |
