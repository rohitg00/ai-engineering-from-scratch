# 回归策略指南

回归预测连续值。正确的方法取决于特征和目标之间的关系、特征数量以及过拟合风险。

## 决策清单

1. 特征和目标之间的关系是否近似线性？
   - 是：从普通线性回归开始
   - 否：尝试多项式特征或非线性模型

2. 相对于样本，您有多少特征？
   - 特征少，样本多：普通线性回归工作正常
   - 特征多，样本少：使用正则化（Ridge 或 Lasso）
   - 特征比样本多：Lasso (L1) 选择特征，或 Ridge (L2) 收缩所有权重

3. 您需要可解释性吗？
   - 是：具有少量特征的线性回归，或 Lasso 用于自动特征选择
   - 否：多项式特征，或转向基于树的模型或神经网络

4. 您的数据集小吗（少于 10,000 行）？
   - 使用正规方程（闭式解）以提高速度
   - 交叉验证对于可靠评估至关重要

5. 您的数据集大吗（数百万行）？
   - 使用随机梯度下降 (SGD) 或小批量梯度下降
   - 由于 O(n^3) 矩阵求逆，正规方程太慢

## 何时使用每种方法

**普通线性回归**：任何回归任务的基线。从这里开始。如果 R-squared 可接受且模型简单，就停在这里。

**多项式回归**：散点图显示曲线，而不是直线。从 2 度开始。仅在验证性能证明合理时增加。度数 >5 几乎总是过拟合。

**Ridge 回归 (L2)**：许多相关特征。所有权重向零收缩，但没有一个完全变为零。当您相信所有特征都有贡献时很好。

**Lasso 回归 (L1)**：许多特征，但您怀疑只有几个重要。Lasso 将不相关特征权重完全设为零，执行自动特征选择。

**Elastic Net**：结合 L1 和 L2 惩罚。当您有许多相关特征并且想要一些特征选择时使用。

## 常见错误

- 在梯度下降之前跳过特征缩放（收敛变得极其缓慢）
- 使用测试集性能来调整超参数（使用验证集或交叉验证）
- 不检查验证误差就拟合高次多项式（训练 R^2 总是随度数增加）
- 忽略残差图（如果残差显示模式，R^2 可能会产生误导）
- 将 R^2 视为唯一指标（检查残差分布、MAE 和领域特定阈值）

## 快速参考

| 方法 | 何时使用 | 正则化 | 特征选择 |
|------|---------|-------|--------|
| OLS | 基线，特征少 | 无 | 手动 |
| Ridge | 特征多，都相关 | L2 (收缩) | 否 |
| Lasso | 特征多，少数相关 | L1 (零化) | 自动 |
| Elastic Net | 许多相关特征 | L1 + L2 | 部分 |
| Polynomial | 非线性关系 | 在顶部添加 Ridge/Lasso | 手动度数选择 |
