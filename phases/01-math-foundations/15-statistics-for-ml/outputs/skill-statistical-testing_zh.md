# ML 的统计检验

如何在比较模型、运行 A/B 实验或验证结果时选择正确的检验。

## 决策清单

1. 你在比较什么？均值、比例、分布还是相关性？
2. 多少组？一个样本 vs 参考、两组还是多组？
3. 观察是配对的（相同测试集、相同折叠）还是独立的？
4. 数据是否正态分布？如果 n < 30 且不明显正态，使用非参数方法。
5. 数据是连续的、有序的还是分类的？
6. 你在运行多少检验？如果多于一个，应用校正。

## 决策树

```text
比较均值？
  两组？
    配对（相同数据分割）？--> 配对 t 检验（或如果不正态使用 Wilcoxon 符号秩）
    独立？--> Welch's t 检验（或如果不正态使用 Mann-Whitney U）
  多组？
    配对？--> 重复测量 ANOVA（或 Friedman 检验）
    独立？--> 单因素 ANOVA（或 Kruskal-Wallis）

比较比例？
  两组？--> 卡方检验或 Fisher 精确检验（小 n）
  多组？--> 卡方检验

比较分布？
  一个分布是参考？--> Kolmogorov-Smirnov 检验
  两个都是经验分布？--> 双样本 KS 检验

测量关联？
  两个都连续，大致正态？--> Pearson 相关
  有序或非正态？--> Spearman 秩相关
  分类 x 分类？--> 卡方独立性检验

运行多个检验？
  应用 Bonferroni 校正：alpha_adjusted = alpha / number_of_tests
  或使用 Holm-Bonferroni（较不保守，仍控制族错误率）
```

## 何时使用每种检验

| 检验 | 数据类型 | 假设 | ML 用例 |
|---|---|---|---|
| 配对 t 检验 | 连续，配对 | 正态差异 | 在相同 k 折分割上比较 2 个模型 |
| Wilcoxon 符号秩 | 连续/有序，配对 | 无（非参数） | 比较 2 个模型，小 k（5-10 折） |
| Welch's t 检验 | 连续，独立 | 大致正态 | 在两个独立数据集上比较模型 |
| Mann-Whitney U | 连续/有序，独立 | 无 | 比较延迟分布 |
| ANOVA | 连续，3+ 组 | 正态，等方差 | 比较多个模型架构 |
| Kruskal-Wallis | 连续/有序，3+ 组 | 无 | 比较多个模型，非正态指标 |
| 卡方 | 分类计数 | 期望计数 >= 5 | 比较类别分布，混淆矩阵 |
| Fisher 精确 | 分类计数 | 小样本 | 罕见事件比较 |
| KS 检验 | 连续 | 无 | 检查预测是否遵循预期分布 |
| Bootstrap CI | 任何统计量 | 无 | AUC、F1、任何指标的置信区间 |
| McNemar 检验 | 配对二元 | 无 | 在相同测试集上比较两个分类器 |

## 模型比较配方

1. 在运行实验前定义指标和显著性水平（alpha = 0.05）。
2. 在相同的 k 折交叉验证分割上运行两个模型（k = 5 或 10）。
3. 收集配对分数：(a_1, b_1), (a_2, b_2), ..., (a_k, b_k)。
4. 计算差异：d_i = b_i - a_i。
5. 运行配对检验（k <= 10 使用 Wilcoxon，k > 10 或正态差异使用配对 t 检验）。
6. 报告：p 值、均值差异、95% 置信区间、效应量（Cohen's d）。
7. 如果 p < alpha 且效应量有意义，差异是真实的且值得采取行动。

## 常见错误

- 数据配对时使用独立检验。如果两个模型在相同的测试折叠上评估，你必须使用配对检验。独立检验丢弃配对并失去统计功效。
- 报告 p < 0.05 而不报告效应量。统计显著的 0.1% 准确率提升不值得部署。始终计算 Cohen's d 或原始均值差异。
- 在不同测试集上比较模型。测试集必须对两个模型相同。不同的测试集使比较无意义。
- 运行 20 次比较并报告最好的一个而不进行 Bonferroni 校正。在 alpha = 0.05 下进行 20 次检验，你期望 1 次假阳性。
- 在不平衡数据上使用准确率。在 99% 多数类上，一个平凡分类器达到 99%。使用 F1、精确率-召回率 AUC 或 Matthews 相关系数。
- 将交叉验证折叠视为独立样本。它们共享训练数据，这违反了独立性假设。校正的重采样 t 检验考虑了这一点。

## 快速参考：效应量解释

| Cohen's d | 解释 |
|---|---|
| 0.2 | 小效应 |
| 0.5 | 中等效应 |
| 0.8 | 大效应 |
| > 1.0 | 非常大效应 |

| 报告内容 | 原因 |
|---|---|
| p 值 | 差异是真实的吗？ |
| 置信区间 | 差异可能有多大？ |
| 效应量（Cohen's d） | 差异有意义吗？ |
| 样本量（n 或 k 折） | 我们能信任结果吗？ |
