# ML信息论

何时在机器学习系统中使用熵、交叉熵、KL 散度和互信息。

## 决策清单

1. 测量单个分布中的不确定性？使用 **熵**。
2. 测量模型近似真实标签的程度？使用 **交叉熵**（这是您的分类损失）。
3. 测量两个分布之间的距离？使用 **KL 散度**。
4. 检查两个变量是否相关？使用 **互信息**。
5. 报告语言模型质量？使用 **困惑度**（交叉熵的指数）。
6. 将一个模型蒸馏到另一个？最小化从教师到学生的 **KL 散度**。

## 何时使用每个度量

| 度量 | 公式 | 用例 | ML应用 |
|---|---|---|---|
| 熵 H(P) | -sum(p log p) | 这个分布有多不确定？ | 数据复杂性，最大熵模型 |
| 交叉熵 H(P,Q) | -sum(p log q) | 模型 Q 预测真实 P 的好坏程度？ | 分类损失，语言模型损失 |
| KL散度 D(P\|\|Q) | sum(p log(p/q)) | P 和 Q 有多不同？ | VAE 损失（ELBO），知识蒸馏，RLHF |
| 互信息 I(X;Y) | H(X) - H(X\|Y) | Y 告诉我们多少关于 X 的信息？ | 特征选择，表示学习 |
| 困惑度 | exp(H(P,Q)) 或 2^H | 模型有多困惑？ | 语言模型评估 |
| 条件熵 H(X\|Y) | -sum(p(x,y) log p(x\|y)) | 在知道 Y 后 X 中的剩余不确定性 | 特征信息量 |

## 关键关系

```
交叉熵 = 熵 + KL散度
H(P, Q) = H(P) + D_KL(P || Q)

由于 H(P) 在训练期间是常数：
  最小化交叉熵 = 最小化 KL散度

互信息 = 熵 - 条件熵
I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)

困惑度 = exp(以 nats 为单位的交叉熵)
       = 2^(以 bits 为单位的交叉熵)
```

## 快速参考：公式和单位

| 公式 | 位（log 底 2） | 奈特（log 底 e） |
|---|---|---|
| 信息：-log(p) | -log2(p) | -ln(p) |
| 熵：-sum(p log p) | bits | nats |
| 1 nat = | 1.4427 bits | 1 nat |
| PyTorch 默认 | -- | nats |
| 信息论论文 | bits | -- |

## 解释值

| 熵值 | 含义 |
|---|---|
| 0 | 确定性。一个结果概率为 1。 |
| log(n) | 最大不确定性。n 个结果上的均匀分布。 |
| 低 | 分布是尖峰的。模型有信心。 |
| 高 | 分布是平坦的。模型不确定。 |

| 困惑度值 | 语言模型质量 |
|---|---|
| 1 | 完美预测（在实践中从未发生） |
| 10 | 平均在 ~10 个等可能的 token 中选择 |
| 50 | 标准基准上的 GPT-2 水平 |
| < 10 | 对于代表性良好的领域的最新技术 |

## 常见错误

- 计算 KL 散度并将其视为对称的。D_KL(P||Q) != D_KL(Q||P)。对于对称度量，使用 Jensen-Shannon 散度：JS = 0.5 * KL(P||M) + 0.5 * KL(Q||M)，其中 M = 0.5*(P+Q)。
- 忘记 one-hot 标签的交叉熵简化为 -log(p_true_class)。当真实分布是 one-hot 时，您不需要对所有类别求和。
- 在代码中使用 log 底 2 但报告 nats（或反之亦然）。PyTorch 默认使用自然对数。乘以 log2(e) = 1.4427 将 nats 转换为 bits。
- 计算空或零概率事件的熵。约定：0 * log(0) = 0，因为 lim(p->0) p*log(p) = 0。
- 跨不同词汇表比较困惑度。词汇表大小 50k 且困惑度为 30 的模型与词汇表大小 10k 且困惑度为 30 的模型不直接可比。

## 每个概念在生产 ML 中的出现位置

| 概念 | 您在哪里看到它 |
|---|---|
| 交叉熵损失 | 每个分类模型（nn.CrossEntropyLoss） |
| KL散度 | VAE ELBO、PPO裁剪、知识蒸馏 |
| 熵正则化 | RL中的探索奖励（更高熵 = 更多探索） |
| 互信息 | 特征选择、InfoNCE损失（对比学习） |
| 困惑度 | 语言模型基准（越低越好） |
| 标签平滑 | 用软目标替换 one-hot，减少交叉熵过度自信 |
| 温度缩放 | 在 softmax 之前将 logits 除以 T，控制输出熵 |
