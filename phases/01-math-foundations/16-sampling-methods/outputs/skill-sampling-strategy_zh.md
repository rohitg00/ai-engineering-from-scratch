# 采样策略选择

如何为文本生成、贝叶斯推断、蒙特卡洛估计和训练选择正确的采样方法。

## 决策清单

1. 你在生成输出（文本、图像）还是估计量（积分、期望）？
2. 你能直接从目标分布采样，还是只能评估其密度？
3. 目标分布是离散的还是连续的？
4. 样本空间的维度是多少？低（< 5）、中（5-100）还是高（> 100）？
5. 你需要精确样本还是近似样本？
6. 你需要通过采样操作的梯度吗？

## 何时使用每种方法

| 方法 | 何时使用 | 复杂度 | 精确？ |
|---|---|---|---|
| 直接采样 | 你有 CDF 或可以使用库函数 | 每次采样 O(1) | 是 |
| 逆 CDF | 已知闭式 CDF 逆（指数、柯西） | 每次采样 O(1) | 是 |
| Box-Muller | 需要没有库的正态样本 | 每次采样 O(1) | 是 |
| 拒绝采样 | 可以评估目标 PDF，低维（1-3） | 每次采样 O(1/接受率) | 是 |
| 重要性采样 | 需要期望，不是单个样本 | n 个样本 O(n) | 近似 |
| 分层采样 | 蒙特卡洛估计，想要更低方差 | n 个样本 O(n) | 近似 |
| Metropolis-Hastings | 高维，可以评估未归一化密度 | 每步 O(1) + burn-in | 渐近 |
| Gibbs 采样 | 可以从每个条件分布采样 | 每次完整扫描 O(d) | 渐近 |
| HMC/NUTS | 高维连续，平滑密度 | 每步 O(L * d) | 渐近 |
| 温度采样 | LLM 文本生成，控制创造性 | O(V) 词汇大小 V | N/A |
| Top-k 采样 | LLM 生成，移除不太可能 token | O(V log k) | N/A |
| Top-p（核） | LLM 生成，自适应候选集 | O(V log V) | N/A |
| 重参数化 | 需要通过高斯采样的梯度（VAE） | O(d) | 是 |
| Gumbel-Softmax | 需要通过分类采样的梯度 | O(k) k 个类别 | 近似 |

## LLM 生成设置

| 用例 | 温度 | Top-p | Top-k | 备注 |
|---|---|---|---|---|
| 事实问答 | 0.0（贪婪） | -- | -- | 确定性，无随机性 |
| 代码生成 | 0.2-0.5 | 0.9 | -- | 低创造性，高连贯性 |
| 一般聊天 | 0.7 | 0.9 | -- | 平衡 |
| 创意写作 | 0.9-1.2 | 0.95 | -- | 更高多样性 |
| 头脑风暴 | 1.0-1.5 | 0.95 | -- | 最大多样性，可能失去连贯性 |

温度和 top-p 可以结合。先应用温度（缩放 logits），然后应用 top-p 过滤。

## MCMC 方法选择

| 属性 | Metropolis-Hastings | Gibbs | HMC/NUTS |
|---|---|---|---|
| 维度 | 任何 | 任何（最佳 < 100） | 高（100+） |
| 需要条件分布 | 否 | 是 | 否 |
| 需要梯度 | 否 | 否 | 是 |
| 接受率 | 调整到 ~23% | 总是 100% | 调整到 ~65% |
| 相关性 | 高（随机游走） | 中等 | 低 |
| Burn-in | 长 | 中等 | 短 |
| 最适合 | 探索，简单模型 | 共轭模型，贝叶斯网络 | 连续后验，深度概率模型 |

## 常见错误

- 在高维中使用拒绝采样。接受率随维度指数下降。超过 5 维，切换到 MCMC。
- 设置 MCMC 提议方差太高或太低。太高：大多数提议被拒绝，链卡住。太低：所有提议被接受，链移动缓慢。目标 ~23% 接受率用于随机游走 MH。
- 忘记 burn-in。MCMC 的前 N 个样本被起始点偏置。至少丢弃 1000 步（或复杂分布更多）。
- 使用与目标非常不同的提议进行重要性采样。少数样本获得巨大权重，使估计不可靠。监控有效样本量：ESS = (sum w_i)^2 / sum(w_i^2)。
- 对需要确定性输出的任务使用温度 > 0（例如分类、结构化提取）。使用贪婪（T=0）或束搜索。
- 不将 top-p 与温度结合。单独的温度不会从长尾中移除垃圾 token。Top-p 会。
- 通过标准采样操作反向传播。对连续（高斯）使用重参数化技巧，对离散（分类）使用 Gumbel-Softmax。

## 快速参考：方差减少技术

| 技术 | 如何工作 | 方差减少 |
|---|---|---|
| 分层采样 | 将空间分成层，每层采样 | 总是 <= 标准 MC |
| 对偶变量 | 使用 U 和 1-U | 对单调函数有效 |
| 控制变量 | 减去已知均值变量 | 与相关性成比例 |
| 重要性采样 | 从更好的提议重新加权样本 | 取决于提议质量 |
| 拉丁超立方 | 独立分层每个维度 | 高维中比分层更好 |
