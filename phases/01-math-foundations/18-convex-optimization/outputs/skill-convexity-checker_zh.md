# 凸性检查器

如何验证优化问题是否是凸的，以及如何处理答案。

## 决策清单

1. 目标函数是凸的吗？（检查 Hessian 半正定性或使用组合规则。）
2. 所有不等式约束的形式是否为 g_i(x) <= 0，其中每个 g_i 是凸的？
3. 所有等式约束都是仿射（线性）的吗？
4. 如果三个都是是，问题是凸的。使用具有收敛保证的凸求解器。
5. 如果有任何否，问题是非凸的。使用 SGD/Adam 并接受局部最优。

## 如何测试函数的凸性

| 测试 | 适用于 | 方法 |
|---|---|---|
| 二阶导数 >= 0 | 标量函数 f(x) | 计算 f''(x)。如果 f''(x) >= 0 对所有 x，凸。 |
| Hessian 是 PSD | 多变量函数 f(x) | 计算 H(x)。如果所有特征值 >= 0 处处，凸。 |
| 定义测试 | 任何函数 | 检查 f(tx + (1-t)y) <= t*f(x) + (1-t)*f(y) 对采样 x, y, t。 |
| 组合规则 | 组合函数 | 见下表。 |
| 限制到直线 | 多变量 f | f 是凸的当且仅当 g(t) = f(x + tv) 对 t 凸对所有 x, v。 |

## 组合规则（保持凸性）

| 操作 | 结果 |
|---|---|
| f + g（都凸） | 凸 |
| c * f（c > 0，f 凸） | 凸 |
| max(f, g)（都凸） | 凸 |
| f(Ax + b) f 凸 | 凸 |
| g(f(x)) g 凸非减且 f 凸 | 凸 |
| g(f(x)) g 凸非增且 f 凹 | 凸 |
| 凸函数的和 | 凸 |
| 凸函数的点态上确界 | 凸 |

## 常见 ML 目标：凸还是不凸？

| 目标 | 凸？ | 原因 |
|---|---|---|
| MSE: (1/n) sum(y - Xw)^2 | 是 | w 中的二次，Hessian = (2/n) X^T X 是 PSD |
| Logistic loss: sum(log(1 + exp(-y_i * w^T x_i))) | 是 | 凸函数的和（log-sum-exp 族） |
| Hinge loss: sum(max(0, 1 - y_i * w^T x_i)) | 是 | 凸（线性）函数的 max |
| L2 正则化: lambda * ||w||^2 | 是 | 二次，Hessian = 2*lambda*I |
| L1 正则化: lambda * ||w||_1 | 是 | 绝对值的和（凸但不可微） |
| Ridge regression: MSE + L2 | 是 | 两个凸函数的和 |
| LASSO: MSE + L1 | 是 | 两个凸函数的和 |
| Elastic net: MSE + L1 + L2 | 是 | 凸函数的和 |
| SVM（原始）: hinge + L2 | 是 | 凸函数的和 |
| Cross-entropy with softmax | 是（在 logits 中） | Log-sum-exp 是凸 |
| Neural network（任何损失） | 否 | 非线性激活创建非凸组合 |
| k-means 目标 | 否 | 离散分配步骤 |
| Matrix factorization: ||X - UV^T||^2 | 否 | U 和 V 双线性 |
| GAN loss | 否 | Minimax，生成器非凸 |
| Contrastive loss (InfoNCE) | 否 | 负样本指数比率的 log |

## 基于凸性的求解器选择

| 问题类型 | 求解器 | 收敛保证 |
|---|---|---|
| 凸，平滑，无约束 | 梯度下降 | O(1/k) 到全局最小值 |
| 凸，平滑，无约束 | L-BFGS | 超线性到全局最小值 |
| 凸，平滑，无约束 | Newton 方法 | 最小值附近二次（如果 Hessian 可处理） |
| 凸，平滑，约束 | 内点法 | 多项式时间 |
| 凸，非平滑（L1） | 近端梯度 / ISTA | O(1/k) 到全局最小值 |
| 凸，非平滑（L1） | ADMM | 灵活，处理约束 |
| 凸，二次 | 共轭梯度 | n 步精确 |
| 非凸，平滑 | SGD / Adam | 收敛到局部最小值 |
| 非凸，平滑 | SGD + 重启 | 平均更好的局部最小值 |
| 非凸，平滑 | 过参数化 + SGD | 平坦最小值，良好泛化 |

## 常见错误

- 假设问题因为损失函数是凸的就是凸的。损失必须在你要优化的参数中是凸的。交叉熵在 logits 中是凸的，但从输入到 logits 的完整神经网络映射是非凸的。
- 在非凸问题上使用 Newton 方法。Hessian 可能有负特征值，导致 Newton 向鞍点或最大值移动而不是最小值。
- 忘记 L1 正则化使目标在零点不可微。标准梯度下降效果不好。使用近端梯度下降或次梯度方法。
- 通过形成 A^T A 平方条件数。如果你需要求解最小二乘问题且 A 病态，使用 QR 或 SVD 而不是正规方程。
- 不检查就声明问题非凸。许多 ML 问题（线性模型、SVM、逻辑回归）是凸的，受益于更强的求解器。

## 快速测试：我的问题是凸的吗？

```
1. 写出目标：最小化 f(w) 受约束
2. 对于 f(w) 中的每一项：
   - 它是具有 PSD 矩阵的二次吗？-> 凸
   - 它是范数吗？-> 凸
   - 它是 log-sum-exp 吗？-> 凸
   - 它涉及 w 非线性（sigmoid(w), w1*w2）？-> 可能非凸
3. 所有约束都是线性或凸不等式吗？
4. 如果所有项都是凸的且约束是凸/线性的 -> 问题是凸的
5. 如果有任何项是非凸的 -> 问题是非凸的
```
