# 顶点项目 17 —— 个人 AI 导师（自适应、多模态、带记忆）

> Khanmigo（Khan Academy）、Duolingo Max、Google LearnLM / Gemini for Education、Quizlet Q-Chat 和 Synthesis Tutor 都在 2026 年大规模交付了自适应多模态辅导。共同形态是苏格拉底式策略（从不直接给出答案）、每次交互后更新的学习者模型（贝叶斯知识追踪风格）、语音 + 文本 + 拍照数学输入、课程图检索、间隔重复调度，以及针对年龄适宜内容的严格安全过滤器。顶点项目是交付一个学科特定导师（K-12 代数或入门 Python），与 10 名学习者进行为期两周的效能研究，并通过内容安全审计。

**类型：** 顶点项目
**语言：** Python（后端、学习者模型）、TypeScript（Web 应用）、SQL（通过 Postgres + Neo4j 的课程图）
**先决条件：** Phase 5（NLP）、Phase 6（语音）、Phase 11（LLM 工程）、Phase 12（多模态）、Phase 14（智能体）、Phase 17（基础设施）、Phase 18（安全）
**涉及阶段：** P5 · P6 · P11 · P12 · P14 · P17 · P18
**时间：** 30 小时

## 问题

自适应辅导过去是教育科技研究的小众领域。到 2026 年，它已成为消费产品。Khanmigo 部署在美国大多数学区。Duolingo Max 达到数千万月活用户。Google 的 LearnLM / Gemini for Education 为 Google Classroom 提供辅导。Quizlet Q-Chat 与抽认卡并存。Synthesis Tutor 以好奇儿童导师的身份走红。共同元素：多模态输入（打字、说话、拍摄方程）、苏格拉底式教学法（先问后解释）、每次交互后更新的学习者模型，以及严格的年龄适宜安全性。

你将为特定群体构建其中之一。测量标准是一项实际效能研究：10 名学习者两周内的前测和后测分数。语音循环必须感觉自然（顶点项目 03 子栈）。记忆必须尊重隐私。安全过滤器必须通过 K-12 的 COPPA 感知红队。

## 概念

四个组件。**导师策略**是苏格拉底式循环：当学习者要求答案时，策略提出一个引导性问题；当他们答对时，进入下一个概念；当他们卡住时，提供一个支架式提示。**学习者模型**是贝叶斯知识追踪（或简单变体），每次交互后更新每个课程节点的掌握概率。**课程图**是 Neo4j 概念图，带先决条件边；策略遍历图以选择下一个概念。**记忆**是情景 + 语义存储（agentmemory 风格），保存过去的交互、错误和偏好。

用户体验是多模态的。文本输入用于打字答案。通过 LiveKit + Whisper 的语音输入（重用顶点项目 03）。通过 dots.ocr 或 PaliGemma 2 的照片输入用于数学问题。通过 Cartesia Sonic-2 的语音输出。安全使用 Llama Guard 4 加上年龄适宜过滤器（阻止成人内容、暴力、自残）和 COPPA 感知记忆保留策略。

效能研究是可交付成果。10 名学习者、前测和后测、两周。报告学习增益差异和置信区间。与非自适应基线（相同内容线性交付，无导师策略）进行比较。

## 技术栈

- 学科选择：K-12 代数或入门 Python（选择一个深入）
- 导师策略：LangGraph 上的 Claude Sonnet 4.7（带提示缓存）
- 学习者模型：贝叶斯知识追踪（经典）或 FSRS 用于间隔
- 课程图：Neo4j 概念 + 先决条件边 + OER 内容
- 记忆：agentmemory 风格持久向量 + 情景 + 语义存储
- 语音：LiveKit Agents 1.0 + Cartesia Sonic-2（重用顶点项目 03 子栈）
- 拍照数学：dots.ocr 或 PaliGemma 2 用于方程识别
- 安全：Llama Guard 4 + 自定义年龄适宜过滤器
- 评估：Bloom 级别问题生成、前/后测工具、效能研究工具

## 构建它

1. **课程图。** 构建一个 50-150 概念节点的 Neo4j（例如，从"数轴"到"二次公式"的 K-12 代数），带先决条件边。每个节点附加 OER 内容（Open Textbook、OpenStax）。

2. **学习者模型。** 用先验初始化贝叶斯知识追踪：猜测、失误、学习率。每次交互后更新每概念掌握。每个学习者持久化。

3. **导师策略。** LangGraph，带节点：`read_signal`（学习者的答案是否正确/部分/卡住？）、`select_concept`（遍历课程图选择最高优先级概念）、`scaffold`（苏格拉底提示）、`update_mastery`。

4. **记忆。** 每次交互写入情景存储。错误和偏好提升到语义记忆。COPPA 感知保留策略：1 年后自动删除，家长可访问。

5. **语音路径。** LiveKit Agents 工作者附加到导师策略。ASR 通过 Whisper-v3-turbo。TTS 通过 Cartesia Sonic-2。支持打断（重用顶点项目 03 机制）。

6. **拍照数学路径。** 上传或捕获图像；运行 dots.ocr 或 PaliGemma 2 识别方程；作为结构化输入馈送给导师。

7. **安全。** 每个模型输出通过 Llama Guard 4 + 年龄适宜过滤器（阻止自残、成人内容、暴力）。记忆访问由学习者 ID 范围控制；家长访问删除界面。

8. **效能研究。** 10 名学习者、前测（标准化 30 问题基线）、两周导师交互（每周 3 次会话）、后测。与相同内容的 10 名学习者非自适应基线群体进行比较。

9. **每周进度报告。** 每个学习者，自动生成 PDF 摘要，包含探索的主题、掌握轨迹和推荐的下一步。

## 使用它

```
学习者："我不明白为什么 3x + 6 = 12 意味着 x = 2"
[信号]   卡住
[概念]  '隔离变量'（先决条件：加减等式）
[支架] "你会从两边减去什么数字来开始？"
学习者："6"
[信号]   正确
[掌握]  加减等式：0.62 -> 0.77
[概念]  继续 '隔离变量'
[支架] "很好。现在 3x / 3 等于什么？"
```

## 练习

1. 有和没有自适应学习者模型（随机概念顺序）运行效能研究。报告差异。预期自适应获胜，但大小是有趣的数字。

2. 添加多模态探测：相同概念问题以文本、语音和照片形式交付。测量学习者是否以他们偏好的模态更快收敛。

3. 构建家长仪表板：练习的主题、掌握轨迹、即将到来的概念、安全事件（任何护栏命中）。COPPA 对齐。

4. 添加语言切换模式：导师接受西班牙语输入并以西班牙语教学。测量 X-Guard 覆盖。

5. 压力测试记忆隐私：验证学习者 A 即使通过语音片段重新摄取攻击也无法看到学习者 B 的数据。记录尝试访问并警报。

## 关键术语

| 术语 | 人们说的 | 实际含义 |
|------|----------|----------|
| 苏格拉底策略 | "问，不要倾倒" | 导师提出引导性问题，而非给出答案 |
| 贝叶斯知识追踪 | "BKT" | 每概念掌握概率的经典学习者模型方程 |
| FSRS | "自由间隔重复调度器" | 2024 年间隔重复调度器，优于 SM-2 |
| 课程图 | "概念 DAG" | 带先决条件边的 Neo4j 概念图 |
| 情景记忆 | "每次交互日志" | 每次交互存储以供后续检索 |
| 语义记忆 | "学习模式存储" | 从情景中压缩的错误和偏好 |
| COPPA | "儿童隐私法" | 美国法律，限制 13 岁以下儿童的数据收集 |
